# Sign-up Form

A clean, responsive sign-up form built with HTML and CSS featuring a split-layout design with a full-height image panel and a structured form panel.

## 🔗 Live Preview

[View Site Preview](https://alrayyan2157.github.io/Sign-up-Form/)

---

## 📁 Project Structure

```
sign-up-form/
├── index.html
├── styles.css
├── image.jpg
└── Pangolin/
    └── Pangolin-Regular.ttf
```

---

## ✨ Features

- **Split layout** — full-height image panel on the left, form on the right
- **Custom font** — uses the Pangolin typeface for a friendly, distinctive feel
- **Responsive form validation** — fields highlight in red on invalid input using `:user-invalid`
- **Focus styles** — branded green focus ring on active inputs
- **Clean fieldset structure** — semantic HTML with intro, input, and sign-in sections

---

## 🛠️ Built With

- HTML5
- CSS3 (Flexbox, custom `@font-face`, pseudo-classes)
- [Pangolin font](https://fonts.google.com/specimen/Pangolin) (local)
- Background photo by [Marek Piwnicki](https://unsplash.com/@marekpiwnicki) on [Unsplash](https://unsplash.com)

---

## 🚀 Getting Started

1. Clone or download the repository
2. Make sure `image.jpg` and the `Pangolin/` font folder are in the same directory as `index.html`
3. Open `index.html` in your browser — no build step required
